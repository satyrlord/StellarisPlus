# StellarisPlus -- Portrait Manifest

## Purpose & Scope

Exhaustive manifest for Stellar Legions portrait assets in
StellarisPlus.

This file is split from
[mod_portrait_reference.md](mod_portrait_reference.md) so the main
reference stays readable.

Range notation is inclusive. If a range skips numbers, that gap exists
in the current source files and is preserved here intentionally.

---

## Shared Assets

- Shared entity asset:
  `gfx/models/portraits/sl_shared/_humanoid_portrait_entities.asset`
  - Defines `sl_humanoid_01_entity`
  - Uses `portrait_humanoid_01_mesh`
- Set wiring:
  `common/portrait_sets/zz_sp_portrait_sets.txt`
- Category wiring:
  `common/portrait_categories/slreup_portrait_categories.txt`

### Shared-Root Notes

- `sl_earthgiant` and `sl_diamondgiant` both reference
  `gfx/models/portraits/sl_giant/`.
- `sl_savagedespoiler` references the plural directory
  `gfx/models/portraits/sl_savagedespoilers/`.
