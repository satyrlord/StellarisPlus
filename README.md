# StellarisPlus

A comprehensive Stellaris overhaul mod integrating multiple Workshop mods.

## Dependencies

StellarisPlus requires
**[UI Overhaul Dynamic](https://steamcommunity.com/sharedfiles/filedetails/?id=1623423360)**.
Subscribe to it on the Steam Workshop and enable it alongside StellarisPlus in the launcher.

## Installation

1. Clone this repository into your Stellaris mod folder:

   ```powershell
   git clone https://github.com/satyrlord/StellarisPlus.git "C:\Users\<YourUsername>\Documents\Paradox Interactive\Stellaris\mod\StellarisPlusv2"
   ```

   Replace `<YourUsername>` with your Windows username.

2. Enable **StellarisPlus** in the Stellaris launcher under Mods.

3. Launch the game.
