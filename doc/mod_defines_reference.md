# StellarisPlus — File Types Reference

This document describes the main file types in this mod and where
changes usually go.

---

## File Types And Where Changes Usually Go

- Gameplay data: `common/**.txt` (Paradox script).
  - District definitions: `common/districts/*.txt`
  - Zone definitions: `common/zones/*.txt`
  - Scripted variables (global tunables): `common/scripted_variables/*.txt`
  - Defines overrides (engine constants): `common/defines/*.txt`
  - Inline scripts used by districts: `common/inline_scripts/districts/*.txt`
